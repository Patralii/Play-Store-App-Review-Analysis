# 📱 Play Store App Review Analysis

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on Google Play Store applications and user reviews to identify trends in app ratings, categories, user sentiment, pricing models, and other business insights.

## 🎯 Business Objective
- Analyze app ratings
- Identify the most popular app categories
- Study the relationship between reviews and ratings
- Compare free vs paid applications
- Analyze user sentiment
- Discover correlations among numerical features

## 📊 Dataset
- Google Play Store Apps Dataset
- Google Play Store User Reviews Dataset

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📈 Key Visualizations
- Distribution of App Ratings
- Top 10 Categories by Number of Apps
- Relationship Between Reviews and Ratings
- Distribution of Free and Paid Apps
- Average Rating by Content Rating
- Distribution of User Sentiments
- Correlation Heatmap

## 💡 Key Insights
- Most apps have ratings between 4.0 and 4.5.
- Family and Games are the most competitive categories.
- Apps with more reviews generally maintain high ratings.
- Free applications dominate the Play Store.
- Most user reviews are positive.
- Reviews and installs show the strongest positive correlation.

This project demonstrates how exploratory data analysis can uncover valuable business insights from app and user review data, helping developers make informed product decisions.
